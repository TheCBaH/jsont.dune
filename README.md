# jsont.dune

[![build](https://github.com/TheCBaH/jsont.dune/actions/workflows/build.yml/badge.svg)](https://github.com/TheCBaH/jsont.dune/actions/workflows/build.yml)
[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/TheCBaH/jsont.dune)

Dune build system for [jsont](https://erratique.ch/software/jsont) — declarative JSON data manipulation for OCaml. Wraps the upstream jsont library as a git submodule and builds it with dune.

## Usage

```sh
make
make test
```
